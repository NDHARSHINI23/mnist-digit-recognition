# Handwritten Digit Recognition (MNIST) - AlFIDO Tech Internship Task

## Overview
Developed a Convolutional Neural Network (CNN) to classify handwritten digits (0–9) using TensorFlow/Keras. Deployed as an interactive web app using Streamlit.

## Files
- `app.py` — Streamlit web app
- `mnist_cnn_model.h5` — trained model
- `requirements.txt` — dependencies

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
3. Upload a digit image and view the prediction.

## Skills Demonstrated
- Deep Learning (CNN)
- Image Preprocessing
- Model Deployment with Streamlit
- Python & TensorFlow
